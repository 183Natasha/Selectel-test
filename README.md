# SelectelTest приложение на Angular

## 📋 Описание проекта

Тестовое задание по направлению «Frontend-разработка» от Selectel.
Gриложение на Angular 19+ с двумя страницами и системой подсчета выбранных элементов.

## ✨ Функциональность

- ✅ Две страницы: приветственная и страница с меню
- ✅ Система чекбоксов с числовыми значениями
- ✅ Динамическое отображение в заголовке:
  - Выбранный раздел
  - Количество выбранных элементов
  - Общая сумма значений
- ✅ Сброс счетчиков при переключении разделов
- ✅ Адаптивная верстка

## 🛠 Технологии

- HTML5, CSS3
- TypeScript
- Angular 19+
